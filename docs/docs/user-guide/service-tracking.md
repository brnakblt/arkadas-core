# 🚌 Servis Takip Sistemi

Öğrenci servislerini gerçek zamanlı GPS ile takip edin.

## Özellikler

- **Gerçek Zamanlı Konum**: Servisin anlık konumu harita üzerinde
- **Tahmini Varış**: Duraklara tahmini varış süreleri
- **Bildirimler**: Yaklaşma ve varış bildirimleri
- **Geçmiş**: Güzergah geçmişi

## Harita Görünümü

Servis takip ekranında aşağıdaki öğeleri göreceksiniz:

| Simge | Anlam |
|-------|-------|
| 🚌 | Servis aracı |
| 📍 | Durak noktası |
| 🏠 | Öğrenci adresi |
| 🏫 | Okul/Merkez |

## Veli Erişimi

Veliler mobil uygulama veya web arayüzü üzerinden:

1. Çocuklarının servisini takip edebilir
2. Servis yaklaştığında bildirim alabilir
3. Şoförle iletişime geçebilir

## Şoför Paneli

Şoförler mobil uygulamada:

1. Günlük güzergahı görür
2. Durak sırasını takip eder
3. Öğrenci biniş/iniş kaydı tutar
4. Acil durum bildirimi gönderebilir

## Raporlar

- Günlük sefer raporları
- Aylık kilometre özeti
- Durak bazlı zaman analizi

!!! warning "GPS Gereksinimi"
    Servis takibi için araçlarda GPS cihazı veya şoför telefonunda konum izni gereklidir.
