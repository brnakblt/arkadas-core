# 📅 Program Yönetimi

Ders, terapi ve etkinlik programlarını planlayın.

## Takvim Görünümü

Program modülü farklı görünümler sunar:

- **Günlük**: Tek günün detaylı programı
- **Haftalık**: Hafta geneli görünümü
- **Aylık**: Ay takvimi

## Ders/Seans Ekleme

1. Takvimde boş bir slota tıklayın
2. Formu doldurun:
   - Öğrenci seçimi
   - Eğitimci seçimi
   - Ders tipi (Bireysel/Grup)
   - Başlangıç ve bitiş saati
   - Modül (varsa)
3. **Kaydet** butonuna tıklayın

## MEB Kuralları

Sistem otomatik olarak kontrol eder:

| Kural | Limit |
|-------|-------|
| Günlük max seans | 4 seans |
| Haftalık max bireysel | 8 saat |
| Haftalık max grup | 8 saat |
| Seans süresi | 45 dakika |

!!! warning "Gün İçi Değişiklik"
    MEB mevzuatına göre gün içinde program değişikliği yapılamaz. Değişiklikler bir gün öncesinden yapılmalıdır.

## Telafi Seansları

Yapılamayan dersler için telafi planlamak:

1. İptal edilecek dersi seçin
2. **Telafi Planla** butonuna tıklayın
3. Yeni tarihi seçin (max 24 ay içinde)
4. **Kaydet**

### Telafi Limitleri

- Aylık bireysel telafi: max 12 saat
- Günlük toplam telafi: max 3 saat

## Plan Kopyalama

Önceki haftaları kopyalamak için:

1. **Kopyala** butonuna tıklayın
2. Kaynak haftayı seçin
3. Hedef haftayı seçin
4. **Uygula**

## MEBBİS Aktarımı

Program verilerini MEBBİS'e aktarmak için:

1. İlgili ayı seçin
2. **MEBBİS'e Gönder** butonuna tıklayın
3. İşlem sonucunu kontrol edin
