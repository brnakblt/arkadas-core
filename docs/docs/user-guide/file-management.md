# 📁 Dosya Yönetimi

Nextcloud entegrasyonu ile güvenli dosya paylaşımı.

## Dosya Paneli

Dosyalar modülünde:

- **Belgelerim**: Kişisel dosyalarınız
- **Paylaşılan**: Sizinle paylaşılan dosyalar
- **Öğrenci Dosyaları**: Öğrenci bazlı belgeler
- **Şablonlar**: Hazır form şablonları

## Dosya Yükleme

1. **Yükle** butonuna tıklayın
2. Dosyaları sürükleyip bırakın veya seçin
3. Hedef klasörü belirleyin
4. **Yükle** butonuna tıklayın

### Desteklenen Formatlar

- Belgeler: PDF, DOC, DOCX, XLS, XLSX
- Görseller: JPG, PNG, GIF
- Diğer: ZIP, MP4, MP3

### Boyut Limitleri

- Tek dosya: max 100 MB
- Toplam alan: Kuruma göre değişir

## Belge Düzenleme

OnlyOffice entegrasyonu ile:

1. Dosyaya tıklayın
2. **Düzenle** seçin
3. Tarayıcı içinde düzenleyin
4. Otomatik kaydedilir

!!! tip "İpucu"
    Aynı anda birden fazla kişi aynı dosyayı düzenleyebilir.

## Paylaşım

### Kurum İçi Paylaşım

1. Dosyaya sağ tıklayın
2. **Paylaş** seçin
3. Kişi veya grup seçin
4. İzin seviyesini belirleyin:
   - Görüntüleme
   - Düzenleme
   - Tam erişim

### Veli Paylaşımı

Velilerle güvenli link paylaşımı:

1. Dosyayı seçin
2. **Link Oluştur** seçin
3. Süre ve şifre belirleyin
4. Linki kopyalayıp gönderin

## Öğrenci Dosyaları

Her öğrenci için otomatik klasör:

```
/Öğrenciler
  /Ahmet Yılmaz
    /BEP
    /Raporlar
    /Faturalar
    /Diğer
```

## Yedekleme

- Dosyalar otomatik yedeklenir
- 30 günlük versiyon geçmişi
- Silinen dosyalar 90 gün çöp kutusunda
