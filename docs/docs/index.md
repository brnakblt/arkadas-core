## 📱 Temel Özellikler

### 🚌 Servis Takibi

Öğrenci servislerini gerçek zamanlı GPS ile takip edin. Veliler çocuklarının konumunu anlık olarak görebilir.

[Daha fazla bilgi →](user-guide/service-tracking.md)

### 📋 Yoklama Sistemi

Yüz tanıma teknolojisi ile otomatik yoklama. Günlük ve aylık raporlar oluşturun.

[Daha fazla bilgi →](user-guide/attendance.md)

### 📅 Program Yönetimi

Ders, terapi ve etkinlik programlarını kolayca planlayın ve yönetin.

[Daha fazla bilgi →](user-guide/schedule.md)

### 📁 Dosya Yönetimi

SFTPGo entegrasyonu ile güvenli dosya paylaşımı. OnlyOffice ile belge düzenleme.

[Daha fazla bilgi →](user-guide/file-management.md)

## 👤 Kullanıcı Rolleri

| Rol | Yetkiler |
|-----|----------|
| **Yönetici** | Tam sistem erişimi, kullanıcı yönetimi |
| **Öğretmen** | Yoklama, program görüntüleme, öğrenci bilgileri |
| **Terapist** | Terapi seansları, raporlama |
| **Veli** | Çocuk takibi, mesajlaşma |
| **Şoför** | Servis takibi, konum paylaşımı |
