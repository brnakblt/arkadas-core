# Track Specification: AI Material Generation

## Metadata
- **Track ID:** `ai-material-gen`
- **Title:** AI-Driven Educational Material Generation
- **Status:** Active
- **Owner:** Lead Agent

## Goal
Develop AI tools to automatically generate customized educational materials (drawing templates, sketch exercises, worksheets) based on the goals defined in a student's BEP.

## Scope
- Integration with FastAPI AI service.
- Template generation logic (SVGs, PDFs).
- "Try Now" shortcut integration in Teacher Dashboard.
- Support for DKT (Speech Therapy) and Autism programs.

## Success Criteria
- Teachers can generate a relevant worksheet in under 10 seconds.
- Generated materials align with the student's current educational goals.
