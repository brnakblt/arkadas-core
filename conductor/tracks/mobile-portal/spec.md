# Track Specification: Mobile Portal

## Metadata
- **Track ID:** `mobile-portal`
- **Title:** Parent/Teacher Mobile App
- **Status:** Active
- **Owner:** Lead Agent

## Goal
Establish a native mobile presence for Arkadaş ERP by connecting the Android and iOS forks to the live backend suite.

## Scope
- Mobile API Security (JWT/2FA).
- Real-time Schedule Synchronization.
- Document/Report viewing via Nextcloud WebDAV.
- Unified Branding (Green/Orange) across platforms.

## Success Criteria
- Native apps can authenticate with Strapi users.
- Students and Schedules are fetched via the SDK.
- Apps build successfully in CI/CD (Docker Builders).
