# Track Specification: Teacher Dashboard

## Metadata
- **Track ID:** `teacher-dashboard`
- **Title:** Teacher-Specific Dashboard Experience
- **Status:** Planning
- **Owner:** Lead Agent

## Goal
Transform the general administrative dashboard into a role-aware experience that optimizes teacher workflow.

## Scope
- Role-based redirection in `arkadas-web`.
- New `TeacherOverview` components.
- Student-centric action shortcuts (BEP, Attendance).
- Tablet-optimized layout.

## Success Criteria
- Primary teacher actions accessible within 1-2 taps.
- Seamless integration with existing backend APIs.
