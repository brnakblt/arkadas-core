# Technology Stack - Arkadaş Özel Eğitim ERP

## Frontend
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Context API / Zustand
- **Components:** Lucide React for iconography

## Backend & CMS
- **Headless CMS:** Strapi v5 (Node.js)
- **Language:** TypeScript
- **Database:** PostgreSQL 15

## Microservices (api/)
- **OpenCV API:** Python 3.13, FastAPI, dlib (for facial recognition and BSDK simulation)
- **File Management:** SFTPGo (WebDAV support)
- **Document Editing:** OnlyOffice Document Server

## Infrastructure & DevOps
- **Orchestration:** Docker Compose
- **Monorepo Management:** TurboRepo
- **Task Runner:** npm Workspaces
- **Caching & Sessions:** Redis
- **Tunnels:** Cloudflare Tunnel (cloudflared)
- **Secrets Management:** Infisical

## AI Integration
- **Model:** Google Gemini 2.0 Flash (for BEP generation and chatbot)
